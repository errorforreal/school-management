# School Management API

Simple REST API to add and list schools based on location.

## Tech Stack

- Node.js
- Express.js
- MySQL
- mysql2

---

## Run Locally

### 1. Clone Repository

```bash
git clone <repo-url>
cd school-management/backend
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Create `.env`

```env
PORT=3000

DB_HOST=
DB_PORT=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

---

### 4. Setup Database



---

### 5. Start Server

```bash
npm run dev
```

OR

```bash
npm start
```
